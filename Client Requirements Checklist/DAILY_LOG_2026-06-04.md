**3. Admin Dashboard — Full Fulfillment System Built and Working End-to-End**
This was the largest feature of the day. The Shipment Timeline in the admin dashboard is now fully wired to Shopify — when Scott advances a stage, Shopify updates in real time and the appropriate customer email fires automatically. Here is what was built:

- **`lib/admin/carriers.ts`** — New file containing the complete Shopify carrier list (Canada Post, UPS, FedEx, Purolator, DHL Express, Canpar, USPS, Amazon Logistics, Other) with their Shopify API codes.
- **Carrier dropdown in the modal** — Replaced the plain text input with a proper `<select>` dropdown matching all carriers available in Shopify Admin's "Mark as fulfilled" modal.
- **"Notify customer by email" checkbox** — Was previously disabled (marked Plan 2). Now fully enabled and wired — when checked, Shopify sends the appropriate notification email automatically.
- **`POST /api/admin/orders/[id]/fulfill`** — New API route that handles the Shopify mutations:
  - "Shipped" stage → calls `fulfillmentCreateV2` (creates Shopify fulfillment, triggers Shipping Confirmation email)
  - All subsequent stages → calls `fulfillmentEventCreate` (triggers Out for delivery / Delivered emails if Shopify toggles are ON)
- **Fulfillment event persistence** — Events previously lived only in React local state (lost on page refresh). Fixed by: reading `fulfillments { events }` from Shopify's API on every page load and reconstructing the timeline. "Order confirmed" remains a synthetic event (generated from order creation date). All stages from Shipped onward now persist across page reloads.
- **Shopify Admin API scopes updated** — Required three rounds of scope additions via the dev dashboard to resolve progressive access denied errors: `read/write_merchant_managed_fulfillment_orders`, `read_customers`, `read/write_fulfillments`. App is now at version `acme-admin-dashboard-7`.

Confirmed working end-to-end: advancing stages in the custom admin dashboard now correctly updates Shopify (orders show Fulfilled + Delivered in Shopify Admin) and sends the right emails to the customer at each stage.
