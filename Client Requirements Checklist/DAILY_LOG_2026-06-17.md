**8. Converted All 271 Images from JPG/PNG to WebP Format**

After downloading, wrote a Python script to automatically convert all 271 images from their original JPG and PNG formats into **WebP** — a modern image format that:

- Is significantly **smaller in file size** than JPG/PNG (same visual quality, less data)
- Makes the website **load faster**, especially on pages with many images (like a gallery or portfolio)
- Is now supported by all modern browsers

All 271 converted WebP files are ready to be used in the new acmesign.ca build.
