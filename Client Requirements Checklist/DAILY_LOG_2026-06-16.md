**1. Admin Login & Password Reset — Full Bug Fix**

Resolved all outstanding issues with the admin login, forgot password, and password reset flows:

- **`ADMIN_PASSWORD_HASH` and `ADMIN_EMAIL` were both empty on Vercel** — corrected and redeployed. Admin password is `Acme@2026!Scott`, 3 OTP recipients restored.
- **`.env.local` bcrypt hash mangling** — `$` signs in the hash were being expanded by dotenv. Fixed by wrapping in single quotes: `ADMIN_PASSWORD_HASH='$2b$12$...'`
