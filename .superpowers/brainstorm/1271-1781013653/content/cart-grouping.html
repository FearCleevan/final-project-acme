My recommendation: X (flat list). It requires zero changes to the existing cart components — the cart already handles independent line items. Grouping (Y) requires a new grouping algorithm, more complex rendering, and needs to be kept in sync with Shopify's flat line-item model. For a store at this scale, the flat list is clear enough — the colour name on each row makes it readable.
